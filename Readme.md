简介
---
个人办公环境下，登录各个业务系统有不同的账号(没有单点登录), 经常要打开账号记事本去查对应系统的登录信息，比较麻烦!
所以就写了这么个小工具，提高获取字段信息的效率.

安装
---
```shell
npm i pkout -g
```

使用
---
```bash
# 获取全部数据
pkout

# 获取某项数据
pkout baidu.url

# 查看数据文件位置
pkout file

# 查看帮助
pkout -h

```


注意
---
数据文件用yaml格式书写(*yaml用缩进表示数据结构，可读性很好*)
